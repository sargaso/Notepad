# Notepad
Андроид приложение "Заметки"
## Функции
* Отображение списка заметок.с указанием заголовка и даты создания
* Создание заметки
* Редактирование

## Стэк технологий
* Singltone
* Coroutines
* View Binding
* kotlinx.serialization - Сериализация данных
* Room - Для хранения данных в SQLite
